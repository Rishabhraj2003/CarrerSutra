# CareerSutra 🚀

CareerSutra is an AI-powered placement preparation platform built using MERN and Generative AI.

## Features

- Personalized Roadmap Generator
- Progress Tracking Dashboard
- AI Resume Analyzer
- Job Matching System
- Mock Interview Module
- AI Mentor Chatbot

## Tech Stack

- MongoDB
- Express.js
- React.js
- Node.js
- OpenAI API
